I principali strumenti usati per realizzare le mappe

  • Qgis, per creare, modificare, visualizzare, analizzare e pubblicare le informazioni geospaziali su Windows, Mac, Linux, BSD;

  • Visual Studio Code, è un editor di codice sorgente sviluppato da Microsoft per Windows, Linux e macOS

  • Adobe Dreamweaver, crea e gestisci siti web che garantiscono una resa ottimale

  • Qgis2web, plugin per Qgis per esportare il progetto in un webmap OpenLayers o foglio elettronico. Non è richiesto alcun software lato server;

  • QTiles, plugin per Qgis per per generare tiles raster dal progetto QGIS per livelli di zoom selezionati e le convenzioni di denominazione di piastrelle (Slippy mappa o TMS);

  • Qgis2threejs, il plugin Qgis2threejs esporta i dati del terreno, mappa l'immagine della tela e i dati vettoriali nel browser web;

  • Map Warper, è uno strumento open source per georeferenziare immagini, nato grazie al contributo del progetto spacetime della Biblioteca Comunale di New York, sviluppato, ospitato e mantenuto da Tim Waters;

  • Leaflet, per creare l'interfaccia di web mapping;

  • Leaflet-tileoverlay-mask, per la sovrapposizione delle tile con effetto svg maschera, una sorta di effetto “occhio di bue“ che ci permette di vedere come era la città al tempo in un particolare punto;

  • Leaflet hash, per generare URL che contengono livello di zoom e posizione corrente;

  • Sidebar v2, per creare questa sidebar a scomparsa;

  • Leaflet MousePosition, è un semplice controllo della posizione del mouse. Esso mostra le coordinate geografiche del puntatore del mouse, mentre viene spostato sulla mappa;

  • Leaflet graphicscale, è un semplice barra di scala con effetto animato al cambio dello zoom;

  • Leaflet.Basemaps, consente di creare un controllo dell'interfaccia utente per la scelta della mappa di base utilizzato sulla mappa;

  • Leaflet PolylineMeasure, consente di misurare distanze con linee semplici, sistema metrico (metri, chilometri);

  • Leaflet pattern, plugin per leaflet che consente l'utilizzo di pattern di riempimento;

  • Leaflet zoomhome, aggiunge un pulsante Home al controllo zoom per ripristinare la visualizzazione;

  • Leaflet-easyPrint, un semplice plugin che aggiunge un'icona per stampare la mappa;

  • Leaflet ResizableControl, un semplice pannello di controllo ridimensionabile e scorrimento;

  • Leaflet CountrySelect, un semplice puglin per selezionare il Stato/Regione/Provincia/Comune desiderato;

  • Leaflet Credits, plugin per leaflet per la visualizzazione dei crediti nell'angolo;

  • Leaflet pattern, plugin per leaflet che consente l'utilizzo di pattern di riempimento;

  • WRLD per creare l'interfaccia di 3D web mapping;

  • Btns, è un piccolo modulo CSS che permette di inserire pulsanti reattivi.

  • Menu jQuery, Creating Collapsible Bootstrap Navbars With jQuery - Responsive-collapse.js;

  • UNITE Gallery, Unite Gallery, touch enabled reponsive image and video web gallery width zoom effect.

  • Menu jQuery, jQuery Plugin For Multilevel Bootstrap Dropdown Navigation;

  • Menu jQuery, Responsive Multi-level Off Screen Navigation Plugin For jQuery - cbFlyout.

  • Bxslider-4, The fully-loaded, responsive jQuery content slider.

  • jQuery, EU Cookie Law popups

  • Prognroll, Un plugin jQuery piccolo e leggero che crea una barra di avanzamento della pagina sulla pagina.

  • Tableau, software leader per la business intelligence: analisi veloci, facili, con qualsiasi sorgente di dati, su qualsiasi dispositivo.

  • My Google Maps, crea e condividi mappe personalizzate con Google My Maps

  • uMap, permette di creare mappe, che fanno uso di OpenStreetMap come sfondo, da inserire nel proprio sito in un minuto

  • USGS - EarthExplorer, Modello digitale del terreno

  • Gimp, GNU Image Manipulation Program;

  • Inkscape, disegna in libertà;

  • LibreOffice 6.2, Software libero di produttività personale multipiattaforma prodotto da The Document Foundation, nato dal fork di OpenOffice.org;

  • Icone per le tue mappe
